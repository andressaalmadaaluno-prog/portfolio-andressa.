Aprenda Python com passos simples

Python é uma ótima linguagem para iniciantes e profissionais, com sintaxe clara, vasta comunidade e grandes possibilidades.

O básico de Python

Comece pelos conceitos mais importantes e pratique em pequenos programas.

Variáveis

Use variáveis para armazenar valores como texto, números e listas.

nome = "Andressa"
idade = 20
print(nome, idade)

Condicionais

Decida o que o programa faz com base em condições.

if idade >= 18:
    print("Maior de idade")
else:
    print("Menor de idade")

Laços

Repita ações usando loops for e while.

for i in range(5):
    print(i)

Funções

Organize o código em blocos reutilizáveis.

def saudacao(nome):
    return f"Olá, {nome}!"

print(saudacao("Andressa"))

Exemplo de projeto simples

Vamos criar um programa que lê texto do usuário e conta palavras.

texto = input("Digite uma frase: ")
palavras = texto.split()
print(f"Sua frase tem {len(palavras)} palavras.")

Esse exemplo mostra como receber dados do usuário, processar informações e exibir resultados.
