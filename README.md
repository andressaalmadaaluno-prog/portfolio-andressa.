# Como Programar em Python

Este repositório contém um site estático que ensina como programar em Python de forma simples e prática.

## Arquivos

- `index.html` - página principal do site.
- `style.css` - estilos visuais do site.

## Como usar

1. Abra `index.html` no navegador.
2. Leia os exemplos de código de Python.
3. Experimente copiar e colar os exemplos em um arquivo `.py` e executar com Python.

## Sobre o site

O site apresenta:

- conceitos básicos de Python: variáveis, condicionais, laços e funções.
- um exemplo prático de programa que conta palavras.
- links para recursos de aprendizado adicional.
